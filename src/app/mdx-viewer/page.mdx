# Sample MDX Page

Use this page to verify Cadence MDX components work.

## What is MDX?

MDX allows you to write JSX directly in markdown files. You can use React components alongside markdown content.

## Text Formatting

Text can be **bold**, *italic*, or `inline code`. You can also create lists:

- First item
- Second item
- Third item

And numbered lists:

1. First item with **bold text**
2. Second item with *italic text*
3. Third item with `inline code`

## Code Examples

TypeScript example:

```typescript
interface Workflow {
  id: string;
  name: string;
  status: 'running' | 'completed' | 'failed';
}

const workflow: Workflow = {
  id: 'wf-123',
  name: 'Sample Workflow',
  status: 'running',
};
```

JSON example:

```json
{
  "workflowId": "sample-workflow-123",
  "domain": "cadence-samples",
  "status": "running",
  "startTime": "2025-12-24T10:00:00Z"
}
```
